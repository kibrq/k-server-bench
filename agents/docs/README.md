# Agent Docs

This folder contains evaluator documentation intended for agent integrations and MCP clients.

Available guides:

- [`evaluator/`](./evaluator/EVALUATE.md): released non-legacy evaluator usage and submission contract
- [`legacy-evaluator/`](./legacy-evaluator/EVALUATE.md): released legacy evaluator usage and submission contract
