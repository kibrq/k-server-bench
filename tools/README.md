# Tools

This folder contains the released evaluator entrypoints and supporting benchmark utilities.

Main areas:

- [`evaluator/`](./evaluator/README.md): released non-legacy evaluator
- [`legacy-evaluator/`](./legacy-evaluator/README.md): released legacy evaluator
- [`build_circle_legacy_instance.py`](./build_circle_legacy_instance.py): helper for building legacy circle instances
- [`check_circle_potential_on_the_fly.py`](./check_circle_potential_on_the_fly.py): direct evaluator-style check for canonical potentials

Treat the evaluator code here as benchmark infrastructure, not candidate-editable task material.
